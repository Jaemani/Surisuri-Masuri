# Mobility Reliability Platform Workspace

전동보장구의 모바일 주행 데이터와 수리·점검 이력을 결합하는 신규 신뢰성 관리 플랫폼 저장소입니다.

실제 개발 대상은 [`mobility-reliability-platform/`](./mobility-reliability-platform/)입니다. 기존 `soo-ri`, `soo-ri-admin`, `power_assist_device_helper_backend`와 인계용 `attached` 자료는 로컬 참고 자산이며 이 Git 이력에 포함하지 않습니다.

기존 프로젝트의 공개 이력은 [`techforimpact-archive/TFI_CAMPUS_25Spring_Soori-soori`](https://github.com/techforimpact-archive/TFI_CAMPUS_25Spring_Soori-soori)에서 확인할 수 있습니다. 해당 archive는 과거 성과·요구사항·데이터 구조를 이해하기 위한 참고 자료이며, 신규 제품의 코드·브랜드·배포 런타임 의존성이 아닙니다.

## Git 원칙

- 작성자·GitHub 인증 계정은 `Jaemani`인지 확인한 뒤 커밋합니다.
- 기존 브랜드·코드·첨부 개인정보를 신규 이력에 포함하지 않습니다.
- 기능, 아키텍처, 문서 기반을 작은 논리 단위 커밋으로 나눕니다.
- 실제 제품 변화, 결정, 중대 오류, 사람 대상 리포트, 증거를 각각의 문서 스트림에 기록합니다.

프로젝트 범위와 현재 상태는 [신규 프로젝트 README](./mobility-reliability-platform/README.md)를 확인하세요.
