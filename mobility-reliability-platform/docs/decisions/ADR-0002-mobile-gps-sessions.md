# ADR-0002: 모바일 자체 GPS와 명시적 주행 세션

- 상태: accepted
- 결정일: 2026-07-21

## 맥락

과거 외부 API는 실시간 좌표가 아니라 날짜별 주행거리만 제공했다. 신규 프로젝트는 외부 IoT 센서와 환경에 의존하지 않고 Android와 iPhone 자체 GPS를 사용한다. 다만 휴대폰의 이동이 항상 전동보장구 주행을 의미하지는 않는다.

## 검토한 선택지

1. 항상 켜진 background GPS를 전동보장구 이동으로 간주
2. 사용자가 시작·종료하는 주행 세션만 수집
3. 완전 자동 이동 분류 모델이 세션을 결정
4. 명시적 세션을 기본으로 하고 자동 판별을 보조 신호로 사용

## 결정

선택지 4를 채택한다.

- 접근 가능한 한 번의 동작으로 주행 세션을 시작·종료한다.
- 시작된 세션은 화면이 꺼진 동안에도 OS가 허용하는 범위에서 수집한다.
- 속도·가속도·GPS accuracy 기반 모델은 잘못된 샘플과 다른 이동수단 가능성을 표시한다.
- 모델이 사용자의 세션 기록을 조용히 삭제하지 않는다.

## 결과

- 사용자의 작은 입력이 필요하지만 데이터 귀속의 설명 가능성이 높아진다.
- 자동 판별은 향후 데이터가 쌓일수록 개선할 수 있다.
- iOS와 Android lifecycle 차이를 각각 실기기에서 검증해야 한다.
